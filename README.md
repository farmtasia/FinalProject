# 🌿🌱🌳

${\textsf{\color{green}Farmtasia}}$는 마법을 사용하지 못하는 마법사가 아름다운 숲속 마을에서 살아가는 ${\textsf{\color{green}힐링 농장 경영 시뮬레이션 게임}}$입니다.   
작물과 과일을 재배하고 바다에서 낚시를 즐기며, 따뜻한 마을 주민들과 교류하는 평화로운 일상을 경험해 보세요!
